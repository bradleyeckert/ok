see.c words

help ( <name> -- )

Show stack effect of <name>. Open an HTML page showing more detailed information on <name>.

locate ( [lines] <name> -- )

Show source code of <name>. If [lines] is not supplied (stack is empty), the default is 10 lines. If [lines] is negative, the located word is on the last line.

where ( <name> -- )

Show where <name> is referenced in files.

words ( [substring] -- )

List the definition names in the first word list of the search order. Standard TOOLS word. An optional <substring> filters the results to words containing the substring.

wrds ( wid [substring] -- )

List the definition names in a wordlist. An optional <substring> filters the results to words containing the substring.

see ( <name> -- )

Disassemble <name>. Standard TOOLS word.

dasm ( [ a u ] -- )

Disassemble u code words starting at address a. If the stack is empty, disassemble all (up to cp) starting at address 0.

.ir ( inst -- )

Disassemble inst.

helpprefix ( -- )

List the system command prefixes to open help pages in a web browser.

help_forth: ( "ccc<eol>" -- )

The rest of the line is the new system command prefix for web help.

help_local: ( "ccc<eol>" -- )

The rest of the line is the new system command prefix for local help.

.boiler: ( -- )

Fetch the boilerplate from the target and display it.
