quit.c words

base! ( base -- )

Sets the host base if base > 1.

cd ( ccc<eol> -- )

Sets the current working directory.

vocab ( <name> -- )

Create a wordlist that overwrites the top of the search order.

only ( -- )

Clear the search order to "host host".

empty ( -- )

Restores the dictionary to its startup state. Also halts the VM.

halt ( -- )

Halts the VM.

run ( -- )

Resumes running the VM.

t{ ( -- )

Begin actual part of an assert statement.

-> ( -- )

Begin expected part of an assert statement.

}t ( -- )

End an assert statement.

verbose! ( mask -- )

Sets the options for run-time tracing (0 = none). verbose! accepts a combination of flags, so the following constants are a power of 2.

verbose_color ( -- mask )

Use ANSI color escape sequences. Set to 0 if your terminal does not support color.

verbose_bci ( -- mask )

Display the BCI traffic.

verbose_token ( -- mask )

Display the blank-delimited tokens.

verbose_src ( -- mask )

Display the input remaining after each token.

verbose_source ( -- mask )

Display the entire input line.

verbose_cycles ( -- mask )

Display the cycle count.

verbose_fatal ( -- mask )

If an error occurs, exit the application with the error code.

\. ( ccc<eol> -- )

Echo text until EOL.
